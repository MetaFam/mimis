M誰mis is a collaborative file system stored in IPFS with the context being stored in a P2P swarm of Neo4j instances.


```shell
$ forge test
```

#### Format

```shell
$ forge fmt
```

#### Gas Snapshots

```shell
$ forge snapshot
```

```shell

```

```shell

```

#### Cast

```shell

```

#### Help

```shell
$ forge --help
```
